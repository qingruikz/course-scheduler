# 大学授業スケジュールジェネレーター

大学の授業スケジュールを自動生成する Web アプリケーションです。年度、学期、授業回数、曜日、実施方法などの条件を設定することで、休日を考慮した授業日程を自動生成します。

## 機能

- **年度・学期選択**: 複数年度の学年暦データに対応
- **柔軟な授業設定**:
  - 授業回数: 7 回または 14 回
  - 週の授業回数: 週 1 回または週 2 回
  - 曜日選択: 複数の曜日から選択可能
  - 実施方法: 対面またはオンライン（曜日ごとに設定可能）
- **自動休日処理**: 祝日、振替休日、試験期間、長期休暇などを自動的に休講として処理
- **視覚的な表示**:
  - 授業日程リスト表示
  - カレンダー表示（月別）
  - 学年暦モーダル表示
- **データ出力**:
  - Excel 形式（.xlsx）
  - テキスト形式（.txt）
  - Markdown 形式（.md）
  - JSON 形式（.json）
  - クリップボードへのコピー

## 技術スタック

- **フレームワーク**: Vue 3 (Composition API)
- **言語**: TypeScript
- **ビルドツール**: Vite
- **主要ライブラリ**:
  - `xlsx`: Excel ファイルの生成
  - `@modyfi/vite-plugin-yaml`: YAML ファイルの読み込み
  - `vite-plugin-singlefile`: 単一ファイルビルド対応

## インストール

### 必要な環境

- Node.js: `^20.19.0` または `>=22.12.0`
- パッケージマネージャー: `pnpm`（推奨）または `npm`

### セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

ブラウザで `http://localhost:5173` を開いてアプリケーションにアクセスできます。

## 使用方法

1. **年度を選択**: ドロップダウンから年度を選択します
2. **学期を選択**: 1 学期、2 学期、前期、後期などから選択します
3. **授業回数を設定**: 7 回または 14 回を選択します
4. **週の授業回数を設定**: 週 1 回または週 2 回を選択します
5. **曜日を選択**: 授業を行う曜日を選択します（週 2 回の場合は 2 つの曜日を選択）
6. **実施方法を設定**: 各曜日ごとに対面またはオンラインを選択します
7. **スケジュールを生成**: 「スケジュールを生成」ボタンをクリックします
8. **データを出力**: 生成されたスケジュールを Excel、TXT、Markdown、JSON 形式で出力できます

## データ形式

学年暦データは `src/data/` ディレクトリ内の YAML ファイル（`calendar_YYYY.yaml`）として管理されます。

### YAML ファイルの構造

```yaml
year: 2025

semesters:
  1学期: { start: 2025-04-16, end: 2025-06-10 }
  2学期: { start: 2025-06-13, end: 2025-07-31 }
  前期: { start: 2025-04-16, end: 2025-07-31 }
  後期: { start: 2025-09-19, end: 2026-01-26 }
  # ... その他の学期

events:
  - name: 春休み
    start: 2025-04-01
    end: 2025-04-15
  - name: 憲法記念日
    date: 2025-05-03
  # ... その他の休日・イベント
```

### 新しい年度のデータを追加する方法

1. `src/data/` ディレクトリに `calendar_YYYY.yaml` ファイルを作成します
2. 上記の構造に従って年度、学期、休日情報を記述します
3. アプリケーションを再起動すると、新しい年度が自動的に読み込まれます

## ビルド

### 通常のビルド

```bash
pnpm build
```

ビルド結果は `dist/` ディレクトリに出力されます。

### 単一ファイルビルド

```bash
pnpm build:single
```

すべてのリソースを 1 つの HTML ファイルにまとめたビルドが `dist-single/` ディレクトリに出力されます。この形式は、外部ファイルに依存せずに配布できます。

### プレビュー

```bash
pnpm preview
```

ビルドされたアプリケーションをローカルでプレビューできます。

├── vite.config.ts
