## クイックガイド

### 1. 課程設定とスケジュール生成

左側の**条件設定**パネルで、授業の前提条件を設定します。

| 項目 | 説明 |
|------|------|
| **年度** | 対象となる学年（例: 2026 は 2026年度）。学年暦データに基づいて授業日を計算します。 |
| **科目** | 任意。複数科目を登録し、切り替えて利用できます。設定のエクスポート時に科目ごとの設定をまとめて保存できます。 |
| **学期** | 1学期〜4学期、前期・後期、**夏期集中授業期間**・**春季集中授業期間**から選択。 |
| **授業回数** | 7回 または 14回。 |
| **1週間の授業回数** | 通常学期のみ表示。週1回 または 週2回。 |
| **授業の実施方法等** | 通常学期のみ表示。各スロットで「実施方法」（対面 / オンライン同時双方向 / オンデマンド）、**授業日**（曜日）、**時限**を設定。 |

設定後、**「スケジュールを生成」**ボタンを押すと、右側に**授業日程リスト**と**カレンダー**が表示されます。  
（**夏期集中授業期間**・**春季集中授業期間**を選んだ場合は、授業回数のみ設定し、スケジュール生成後にカレンダー上で日付をクリックして授業日を追加する流れになります。詳しくは[集中授業モード](#集中授業モードでの利用)を参照。）

### 2. 授業スケジュールをカレンダーに追加

生成された授業スケジュールを、iPhone・Googleカレンダー等のカレンダーアプリに追加する手順です。

1. **授業日程リスト**のヘッダーにある**カレンダーアイコン**（オレンジ色）をクリックし、「カレンダーに追加」モーダルを開く。
2. **Step 0（設定内容）**  
   - 科目名、曜日・時限・教室（または集中授業の場合は日付・時限・教室）、リマインド、メモなどを確認・編集。  
   - 「配信日をカレンダーに追加」はオンデマンド配信のスロットごとに、その配信日をカレンダーに追加するかどうかを指定できます。
3. **次へ**を押す。
4. **Step 1（カレンダーの選択）**  
   - 利用するカレンダー（Apple / Google / Outlook / その他）を選択。
5. **次へ**を押す。
6. **Step 2**  
   - 環境（OS・ブラウザ）に応じて、**カレンダーファイルのダウンロード**や**QRコード**・**URLをコピー**が表示されます。  
   - スマートフォンで追加する場合は、表示されたQRコードを読み取るか、URLをコピーしてブラウザで開くと、同じ設定でカレンダーに追加できます。

ダウンロードした `.ics` ファイルを、各カレンダーアプリの「インポート」機能で読み込んでください。

---

### 3. 設定のエクスポート・インポート

現在の**条件設定**（年度・科目・学期・授業回数・実施方法など）を、1つのJSONファイルに保存したり、保存したファイルから復元したりできます。

- **エクスポート**  
  - 条件設定パネル上部の**歯車アイコン**をクリック → **「設定をエクスポート」**を選択。  
  - 科目名を入力している場合、複数科目の設定を1つのファイルにまとめて保存できます。科目名が未入力の場合はエクスポートできません。
- **インポート**  
  - 同じ歯車メニューから**「設定をインポート」**を選択し、保存しておいた `.json` ファイルを選ぶと、設定が復元されます。

---

### 4. 学年暦をカレンダーに追加

祝日・学期区分などの**学年暦イベント**を、カレンダーアプリに追加する機能です。

1. 画面上部ヘッダー右の**メニュー（三本線）**をクリック。
2. **「学年暦をカレンダーに追加」**を選択。
3. モーダルで**種別「学年暦」**を選んだ状態で、以下を設定します。  
   - **含めるイベント**: 祝日、学期開始・終了、休講・補講日、授業実施日などから選択。  
   - **授業実施日の扱い**: 授業日のみ / 休講日のみ / すべて から選択。  
   - **リマインド**、**メモ**（任意）。  
   - イベントはすべて「終日」として出力されます。
4. **次へ**でカレンダー（Apple / Google 等）を選択し、**次へ**でダウンロードまたはQR・URLを取得。

## 詳細機能

### 学年暦の表示・管理画面へ移動

ヘッダー右の**メニュー**から、次の画面に移動できます。

| メニュー項目 | 説明 |
|-------------|------|
| **学年暦をカレンダーに追加** | 上記のとおり、学年暦をICS形式でカレンダーに追加するモーダルを開きます。 |
| **大学公式カレンダー** | 大学の公式カレンダーページを**別タブ**で開きます。 |
| **カレンダー配置管理** | 管理画面（`/admin/calendar-mapper`）へ遷移します。ここでは、学年暦の月ごとの配置や日付グリッドを設定し、JSONでエクスポート・インポートできます。本アプリの「学年暦背景」表示は、ここで設定したレイアウトを読み込みます。 |

---

### 集中授業モードでの利用

**学期**で「**夏期集中授業期間**」または「**春季集中授業期間**」を選ぶと、**集中授業モード**になります。

- **条件設定の違い**  
  - 通常の学期では「授業回数」「1週間の授業回数」「授業の実施方法等（曜日・時限）」を設定しますが、集中授業期間では**授業回数（7回/14回）のみ**を設定します。  
  - 授業日は、カレンダー上で**日付をクリック**して、その日に何限目に授業があるかを追加していく形になります。

- **スケジュールの作り方**  
  1. 年度・科目・学期（夏期/春季集中授業期間）・授業回数を設定。  
  2. **「スケジュールを生成」**を押す。  
  3. 右側に、その学期の期間に合わせたカレンダーが表示され、授業日程リストは空の状態になります。  
  4. カレンダー上の**日付をクリック**すると、「この日に追加する授業」を指定するモーダルが開きます。対面/オンライン、時限などを選んで追加。  
  5. リストに「あと○回」など残り回数が表示されます。必要な回数まで日付をクリックして追加してください。

- **リセット**  
  - 授業日程リスト下の**「リセット」**ボタンで、設定した授業日程をすべてクリアし、最初から設定し直せます。確認ダイアログが表示されます。

- **カレンダー表示のオプション**  
  - **対象学期のみ**  
    - オンにすると、集中授業の対象学期の期間のみカレンダーに表示します。オフだと当年度全体を表示します。  
  - **学年暦背景**・**2列表示**・**授業日・休日マーク**  
    - 通常のスケジュールと同様に利用できます。

- **カレンダーに追加（ICS）**  
  - 授業日程リストのカレンダーアイコンから「カレンダーに追加」を開くと、集中授業用の設定（**日付・時限・教室**）として、各授業日ごとにイベントが1件ずつ出力されます。設定内容の表示も「日付・時限・教室」となり、QRコードやURLで他の端末から開いても同じ内容で追加できます。
