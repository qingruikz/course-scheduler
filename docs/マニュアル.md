# 大学授業スケジュールジェネレーター 使用ガイド

[TOC]

## 1. クイックガイド

1. **スケジュール生成**：左側の**条件設定**パネルで、授業の前提条件を設定し、**「スケジュールを生成」**ボタンを押すと、右側に**授業日程リスト**と**カレンダー**が表示されます。

   ![image-20260303221917474](./image-20260303221917474.png)

2. **授業日程リストのコピー・ダウンロード**：**授業日程リスト**のヘッダーにある**ダウンロードアイコン**（緑色）をクリックし、日程リストをコピー・ダウンロードできます。

   <img src="./image-20260303222009420.png" alt="image-20260303222009420" style="zoom:33%;" />

3. **授業スケジュールをiOSカレンダー等に追加**：**授業日程リスト**のヘッダーにある**カレンダーアイコン**（オレンジ色）をクリックし、「カレンダーに追加」モーダルを開き、簡単に設定したら、画面の案内に従って、普段使っているiPhone・Googleカレンダー等のカレンダーアプリに取り込めます。

   <img src="./image-20260303221747266.png" alt="image-20260303221747266" style="zoom:50%;" />

4. **学年暦をiOSカレンダー等に追加**：画面上部ヘッダー右の**メニュー（三本線）**をクリックし、**「学年暦をカレンダーに追加」**を選択。

   ![image-20260303221820940](./image-20260303221820940.png)

5. **設定のエクスポート・インポート**：現在の**条件設定**（年度・科目・学期・授業回数・実施方法など）を、1つのJSONファイルに保存したり、保存したファイルから復元したりできます。

   <img src="./image-20260303221848763.png" alt="image-20260303221848763" style="zoom:33%;" />

   - **エクスポート**：条件設定パネル上部の**歯車アイコン**をクリック → **「設定をエクスポート」**を選択。
   - **インポート**：同じ歯車メニューから**「設定をインポート」**を選択し、保存しておいた `.json` ファイルを選ぶと、設定が復元されます。

## 2. その他の機能

### 2.1 学年暦の表示

ヘッダー右の**メニュー**から、「大学公式カレンダー」を選ぶと、大学の公式カレンダーページを**別タブ**で開きます。

### 2.2 集中授業モードでの利用

**学期**で「**夏期集中授業期間**」または「**春季集中授業期間**」を選ぶと、**集中授業モード**になります。

- **スケジュールの作り方**：
  1. 年度・科目・学期（夏期/春季集中授業期間）・授業回数を設定。
  2. **「スケジュールを生成」**を押す。
  3. 右側に、その学期の期間に合わせたカレンダーが表示され、授業日程リストは空の状態になります。
  4. カレンダー上の**日付をクリック**すると、「この日に追加する授業」を指定するモーダルが開きます。対面/オンライン、時限などを選んで追加。
  5. リストに「あと○回」など残り回数が表示されます。必要な回数まで日付をクリックして追加してください。
- **リセット**：
  - 授業日程リスト下の**「リセット」**ボタンで、設定した授業日程をすべてクリアし、最初から設定し直せます。確認ダイアログが表示されます。
